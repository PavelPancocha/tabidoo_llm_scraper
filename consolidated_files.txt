# File name: synapto-services-main/docs/structure-and-naming-conventions.md
# Monorepo Structure & Naming Conventions

This monorepo uses a flat structure with clear naming rules for packages and apps.

```text
apps/               → Deployable apps & services (React apps, APIs, cron jobs)
packages/           → Shared libraries, clients, UI components, configs
```

## Package Naming Rules (in `packages/`)

- `core-` - Internal frameworks, base services (`core-api-base`, `core-env`)
- `lib-` - Shared business logic / utilities (`lib-auth`, `lib-db`)
- `client-` - API clients for external services (`client-stripe`, `client-flexibee`)
- `ui-` - Shared React code (components, hooks) (`ui-components`, `ui-hooks`)
- `config-` - Shared config for tools (`config-eslint`, `config-ts`)

## Apps (`apps/`)

No special prefixes needed. Just use clear, purpose-based names:

```text
apps/
├── web            → Public React app
├── admin          → Internal admin panel
├── users-api      → Microservice
├── billing-api    → Microservice
├── cron-jobs      → Scheduled jobs
```

# Using MongoDB Atlas

- https://cloud.mongodb.com/
- https://account.mongodb.com/account/login
- https://www.mongodb.com/docs/

Our shared infrastructure uses **MongoDB Atlas** with a **dedicated cluster**, configured to allow connections via both:

- 🔒 **Private networking (GCP VPC Peering)** from our GKE cluster
- 🌍 **Public internet access** (limited IP ranges) from trusted external servers (e.g. Hetzner VPSes)

## MongoDB Atlas Connection Details

### 📌 Public (Internet-based) Access

For services running **outside GCP** (e.g. Hetzner):

- Use the **public DNS hostname**:

```url
cluster0.jwxehw.mongodb.net
```

- Your IP must be added to the **MongoDB Atlas IP Access List**.

- Example connection string:

```url
mongodb+srv://<username>:<password>@cluster0.jwxehw.mongodb.net/<db>?retryWrites=true&w=majority
```

### 🔐 Private GKE Access (via VPC Peering)

For services deployed inside our **GCP GKE cluster**:

- Use the **private DNS hostname**:

```url
  cluster0-pri.jwxehw.mongodb.net
```

- This routes traffic over **internal GCP VPC peering** (faster, private, and cheaper).
- Do **not** use the public hostname in GKE workloads — it will fail.

- Example connection string:

```url
  mongodb+srv://<username>:<password>@cluster0-pri.jwxehw.mongodb.net/<db>?retryWrites=true&w=majority
```

### Dev & CI Notes

- Credentials should never be hardcoded — use **Kubernetes Secrets**, `.env` files, or your CI/CD secret manager.
- All connections require **TLS** — this is handled automatically by modern MongoDB drivers.
- When in doubt, test connectivity with:

From GKE:

```bash
kubectl run mongo-test \
--image=alpine/mongosh \
--restart=Never -it -- \
mongosh "mongodb+srv://<username>:<password>@cluster0-pri.jwxehw.mongodb.net" --tls
```

From Hetzner or local:

```bash
mongosh "mongodb+srv://<username>:<password>@cluster0.jwxehw.mongodb.net" --tls
```

# File name: synapto-services-main/apps/qrcode-api/README.md
# QR Code API

QR Code Generation API service built with Hono.js and OpenAPI specification.

This API allows you to generate QR codes in various formats (PNG, JPEG, SVG) with customizable options including size, colors, error correction levels, and margins.

## Quick Start

```bash
pnpm run dev
open http://localhost:3000
```

**Examples:**

```bash
# Basic QR code
curl "http://localhost:3000/api/qrcode?data=Hello%20World"

# Custom PNG with blue colors
curl "http://localhost:3000/api/qrcode?data=https://example.com&format=png&size=300&colorDark=%23000080&colorLight=%23E0E0FF"

# SVG format with high error correction
curl "http://localhost:3000/api/qrcode?data=Important%20Data&format=svg&errorCorrectionLevel=H"
```

## OpenAPI Documentation

- **Swagger UI**: http://localhost:3000/openapi/ui
- **OpenAPI JSON**: http://localhost:3000/openapi/openapi.json

# File name: synapto-services-main/apps/alert-bridge-api/README.md
# alert-bridge

Service process errors and warning from no-code tools and create alerts in Jira Alerts.

- `make.com` - create inbound smtp server that accepts emails from make.com and sent to Jira Alerts
- `tabidoo` - create job that reads error logs from tabidoo and sends them to Jira Alerts (application are selected based on api key access)

```
pnpm run dev
open http://localhost:3000
```

## Configuration

### Features

- `FEATURES` – comma-separated list of enabled features. Supported values:
  - `smtp` – enables the Make SMTP server
  - `cron` – enables scheduled jobs (Tabidoo error logs download job etc.)

You can combine multiple features: `FEATURES=smtp,cron`

### Jira Alerts

- `JIRA_ALERTS_API_KEY` (required) – Jira Alerts integration API key.
- `JIRA_ALERTS_MODE` (default: `production`) – controls how Jira Alerts are handled:
  - `production` – sends real alerts to Jira
  - `log` – logs alert data to console (useful for local testing)

### Tabidoo

- `FEATURES=cron` – enables scheduled Tabidoo sync jobs.
- `SYNAPTO_CLOUD_API_TOKEN` (required) – token for Synapto Cloud API.
- `TABIDOO_CLOUD_API_TOKEN` (required) – token for Tabidoo Cloud API.

### Make SMTP Server

- `FEATURES=smtp` – enables SMTP server for receiving and parsing incoming emails.
- `MAKE_API_KEY` (required) – API key for Make.com
- `MAKE_ZONE` (default `eu1.make.celonis.com`) - Make.com zone

## Endpoints

### `GET /`

- health check endpoint
- shows which features are enabled

### `GET /cron/apps`

- returns list of apps that are watched by cron job

## Development

To test email parsing locally, install `swaks` and run `send_make_emails_as_text.sh` located in the scripts folder.

To test Tabidoo error logs via API, use `send_tabidoo_errors.sh`, which includes example payloads and authentication.

To run a cron job locally:

- Set `FEATURES=cron` in your .env file
- Make sure both `SYNAPTO_CLOUD_API_TOKEN` and `TABIDOO_CLOUD_API_TOKEN` are set.
- Start the app normally.

## Documentation

- [Jira Alerts API (Integration Events API)](https://developer.atlassian.com/cloud/jira/service-desk-ops/rest/v1/api-group-integration-events)

name: Build and Push to GitHub Container Registry

on:
  workflow_dispatch:
    inputs:
      service_name:
        description: 'Select service to build'
        required: true
        type: choice
        options:
          - '---'
          - alert-bridge-api
          - example-ssr
          # Add more services as needed

jobs:
  validate-input:
    runs-on: ubuntu-latest
    steps:
      - name: Check if service was selected
        if: github.event.inputs.service_name == '---'
        run: |
          echo "::error::No service was selected. Please select a valid service."
          exit 1

  build-and-push:
    runs-on: ubuntu-latest

    permissions:
      contents: read
      packages: write

    steps:
      - name: Checkout code
        uses: actions/checkout@v4

      - name: Login to GitHub Container Registry
        uses: docker/login-action@v3
        with:
          registry: ghcr.io
          username: ${{ github.actor }}
          password: ${{ secrets.GITHUB_TOKEN }}

      - name: Extract metadata for Docker
        id: meta
        uses: docker/metadata-action@v5
        with:
          images: ghcr.io/${{ github.repository_owner }}/${{ github.event.inputs.service_name }}
          tags: |
            type=sha,format=short
            latest

      - name: Build and push Docker image
        uses: docker/build-push-action@v5
        with:
          context: .
          file: ./apps/${{ github.event.inputs.service_name }}/Dockerfile
          push: true
          tags: ${{ steps.meta.outputs.tags }}
          labels: ${{ steps.meta.outputs.labels }}

name: Validate

on:
  push:
    branches:
      - main
  pull_request:
    branches:
      - main

jobs:
  validate:
    name: Validate Code
    runs-on: ubuntu-latest

    steps:
      - name: Checkout code
        uses: actions/checkout@v4
        with:
          fetch-depth: 2

      - name: Setup Node.js
        uses: actions/setup-node@v4
        with:
          node-version: "22"

      - name: Install pnpm
        uses: pnpm/action-setup@v3
        with:
          version: 10
          run_install: false

      - name: Get pnpm store directory
        shell: bash
        run: |
          echo "STORE_PATH=$(pnpm store path --silent)" >> $GITHUB_ENV

      - name: Setup pnpm cache
        uses: actions/cache@v4
        with:
          path: ${{ env.STORE_PATH }}
          key: ${{ runner.os }}-pnpm-store-${{ hashFiles('**/pnpm-lock.yaml') }}
          restore-keys: |
            ${{ runner.os }}-pnpm-store-

      - name: Install dependencies
        run: pnpm install

      - name: Cache Turbo
        uses: actions/cache@v4
        with:
          path: .turbo
          key: ${{ runner.os }}-turbo-${{ github.sha }}
          restore-keys: |
            ${{ runner.os }}-turbo-

      - name: Lint
        run: pnpm run lint

      - name: Build
        run: pnpm run build

      - name: Run tests
        run: pnpm run test

# File name: synapto-services-main/scripts/create-package.js
const fs = require('fs');
const path = require('path');
const { execSync } = require('child_process');

const args = process.argv.slice(2);
if (args.length !== 1) {
  console.error('Usage: pnpm run create-package <new-package-name>');
  process.exit(1);
}

const newPackageName = args[0];
const templatePath = path.join(__dirname, '../packages/lib-example');
const newPackagePath = path.join(__dirname, '../packages', newPackageName);

if (fs.existsSync(newPackagePath)) {
  console.error(`Error: Directory ${newPackageName} already exists.`);
  process.exit(1);
}

// Copy template directory, omitting specific folders
fs.cpSync(templatePath, newPackagePath, {
  recursive: true,
  filter: (src) => {
    const baseName = path.basename(src);
    return !['node_modules', 'dist', '.turbo'].includes(baseName);
  },
});

// Update package.json
const packageJsonPath = path.join(newPackagePath, 'package.json');
const packageJson = JSON.parse(fs.readFileSync(packageJsonPath, 'utf-8'));
packageJson.name = `@repo/${newPackageName}`;
fs.writeFileSync(packageJsonPath, JSON.stringify(packageJson, null, 2));

// Update README.md
const readmePath = path.join(newPackagePath, 'README.md');
if (fs.existsSync(readmePath)) {
  let readmeContent = fs.readFileSync(readmePath, 'utf-8');
  readmeContent = readmeContent.replace(/^# lib-example/i, `# ${newPackageName}`);
  fs.writeFileSync(readmePath, readmeContent);
}

// Install dependencies
console.log(`Installing dependencies for ${newPackageName}...`);
execSync('pnpm install', { cwd: newPackagePath, stdio: 'inherit' });

console.log(`New package ${newPackageName} created successfully at ${newPackagePath}`);

import { Hono } from 'hono';
import { describeRoute } from 'hono-openapi';
import { validator } from 'hono-openapi/zod';
import {
  sayDescription,
  sayParamSchema,
  sumDescription,
  sumRequestBodySchema,
} from '../schemas/example.js';
import type SumService from '../services/SumService.js';

export default function createExampleHandler(sumService: SumService) {
  const app = new Hono();

  app.get(
    '/:name',
    describeRoute(sayDescription),
    validator('param', sayParamSchema),
    async (c) => {
      const { name } = c.req.valid('param');
      return c.json({
        hello: name,
      });
    }
  );

  app.post(
    '/sum',
    describeRoute(sumDescription),
    validator('json', sumRequestBodySchema),
    async (c) => {
      const { a, b } = c.req.valid('json');
      return c.json({
        sum: sumService.add(a, b),
      });
    }
  );

  return app;
}

import { Hono } from 'hono';
import { describeRoute } from 'hono-openapi';
import { validator } from 'hono-openapi/zod';
import { qrCodeDescription, qrCodeQuerySchema } from '../schemas/qrcode.js';
import type QRCodeService from '../services/QRCodeService.js';

export default function createQRCodeHandler(qrCodeService: QRCodeService) {
  const app = new Hono();

  app.get(
    '/',
    describeRoute(qrCodeDescription),
    validator('query', qrCodeQuerySchema),
    async (c) => {
      try {
        const queryParams = c.req.valid('query');

        const result = await qrCodeService.generateQRCode({
          data: queryParams.data,
          format: queryParams.format || 'png',
          size: queryParams.size || 200,
          colorDark: queryParams.colorDark || '#000000',
          colorLight: queryParams.colorLight || '#FFFFFF',
          errorCorrectionLevel: queryParams.errorCorrectionLevel || 'M',
          margin: queryParams.margin || 1,
        });

        c.header('Content-Type', result.contentType);

        if (result.contentType === 'image/svg+xml') {
          return c.body(result.data as string);
        } else {
          return c.body(result.data as Buffer);
        }
      } catch (error) {
        console.error('Error generating QR code:', error);
        return c.json(
          { error: error instanceof Error ? error.message : 'Failed to generate QR code' },
          500
        );
      }
    }
  );

  return app;
}

import * as QRCode from 'qrcode';

export interface QRCodeOptions {
  data: string;
  format: 'png' | 'jpeg' | 'svg';
  size: number;
  colorDark: string;
  colorLight: string;
  errorCorrectionLevel: 'L' | 'M' | 'Q' | 'H';
  margin: number;
}

export default class QRCodeService {
  async generateQRCode(
    options: QRCodeOptions
  ): Promise<{ data: Buffer | string; contentType: string }> {
    const baseOptions = {
      errorCorrectionLevel: options.errorCorrectionLevel,
      margin: options.margin,
      color: {
        dark: options.colorDark,
        light: options.colorLight,
      },
      width: options.size,
    };

    try {
      let qrCodeOutput: Buffer | string;
      let contentType: string;

      switch (options.format) {
        case 'png':
          qrCodeOutput = await QRCode.toBuffer(options.data, {
            ...baseOptions,
            type: 'png',
          } as QRCode.QRCodeToBufferOptions);
          contentType = 'image/png';
          break;
        case 'jpeg':
          // For JPEG, we generate PNG buffer and then would need to convert
          // For simplicity, let's generate PNG instead of JPEG
          qrCodeOutput = await QRCode.toBuffer(options.data, {
            ...baseOptions,
            type: 'png',
          } as QRCode.QRCodeToBufferOptions);
          contentType = 'image/png';
          break;
        case 'svg':
          qrCodeOutput = await QRCode.toString(options.data, {
            ...baseOptions,
            type: 'svg',
          } as QRCode.QRCodeToStringOptions);
          contentType = 'image/svg+xml';
          break;
        default:
          throw new Error(`Unsupported format: ${options.format}`);
      }

      return {
        data: qrCodeOutput,
        contentType,
      };
    } catch (error) {
      throw new Error(
        `Failed to generate QR code: ${error instanceof Error ? error.message : 'Unknown error'}`
      );
    }
  }
}

import { defaultLogger } from '@repo/core-api';
import type { Context } from 'hono';

export function tryOrReturnError(handler: (c: Context) => Promise<Response>) {
  return async (c: Context): Promise<Response> => {
    try {
      return await handler(c);
    } catch (error) {
      if (error instanceof Error) {
        defaultLogger.error(error.message);
        return c.json({ error: error.message }, 400);
      }
      defaultLogger.error('Unknown error occurred', error);
      return c.json({ error: 'Unknown error' }, 400);
    }
  };
}

import { TabidooClient } from '@repo/client-tabidoo';
import { describeRoute } from 'hono-openapi';
import {
  appDescription,
  appDetailDescription,
  appLLMScriptsDescription,
  appTypescriptDefinitionsDescription,
} from '../schemas/apps.js';

import { findTabidooInstanceForToken } from '../utils/findTabidooInstanceForToken.js';
import { getAndFormatCodeForApp } from '../utils/getAndFormatCodeForApp.js';
import { tryOrReturnError } from './base.js';

async function getClientForToken(c: Context): Promise<TabidooClient> {
  const token = c.req.header('token');
  if (!token) {
    throw new Error('Missing token header');
  }

  const client = await findTabidooInstanceForToken(token);
  if (!client) {
    throw new Error('Invalid token');
  }

  return client;
}

export default function createAppsHandler() {
  const app = new Hono();

  app.get(
    '/',
    describeRoute(appDescription),
    tryOrReturnError(async (c) => {
      const client = await getClientForToken(c);
      const apps = await client.apps.all();
      return c.json(apps);
    })
  );

  app.get(
    '/:appId',
    describeRoute(appDetailDescription),
    tryOrReturnError(async (c) => {
      const client = await getClientForToken(c);
      const { appId } = c.req.param();
      const app = await client.apps.get(appId);
      return c.json(app);
    })
  );

  app.get(
    '/:appId/tsd',
    describeRoute(appTypescriptDefinitionsDescription),
    tryOrReturnError(async (c) => {
      const client = await getClientForToken(c);
      const { appId } = c.req.param();
      const scripts = await client.apps.getTypeScriptDefinition(appId);
      c.header('Content-Type', 'text/plain');
      return c.text(scripts.content);
    })
  );

  app.get(
    '/:appId/llm/scripts',
    describeRoute(appLLMScriptsDescription),
    tryOrReturnError(async (c) => {
      const client = await getClientForToken(c);
      const { appId } = c.req.param();
      const content = await getAndFormatCodeForApp(client, appId);
      c.header('Content-Type', 'text/plain');
      return c.text(content);
    })
  );

  return app;
}

import type { TabidooClient } from '@repo/client-tabidoo';
import { createTabidooClient, tabidooInstancesBaseUrls } from '@repo/client-tabidoo';
import { defaultLogger } from '@repo/core-api';

const logger = (message: string, data?: object) => defaultLogger.debug(message, data);

export async function findTabidooInstanceForToken(token: string): Promise<TabidooClient | null> {
  try {
    const synaptoClient = createTabidooClient(tabidooInstancesBaseUrls.synapto, token, logger);
    await synaptoClient.users.me();
    return synaptoClient;
  } catch {
    // Ignore error and try next instance
  }

  try {
    const tabidooClient = createTabidooClient(tabidooInstancesBaseUrls.tabidoo, token, logger);
    await tabidooClient.users.me();
    return tabidooClient;
  } catch {
    // Ignore error
  }

  return null;
}

import {
  extractExecutableCode,
  formatExtractedCodeForLLM,
  type TabidooClient,
} from '@repo/client-tabidoo';
import {
  extractExecutableCodeFromCustomScripts,
  extractExecutableCodeFromWorkflows,
} from '@repo/client-tabidoo/utils/extractScripts';

export async function getAndFormatCodeForApp(client: TabidooClient, appId: string) {
  const [app, customScriptsTable, workflowsTable] = await Promise.all([
    client.apps.get(appId),
    (async () => {
      try {
        return await client.apps.getCustomScripts(appId);
      } catch {
        return null; // Handle case where no custom scripts exist
      }
    })(),
    (async () => {
      try {
        return await client.apps.getWorkflows(appId);
      } catch {
        return null; // Handle case where no workflows exist
      }
    })(),
  ]);

  const scripts = extractExecutableCode(app);
  const workflows = workflowsTable ? extractExecutableCodeFromWorkflows(workflowsTable) : [];
  const customScripts = customScriptsTable
    ? extractExecutableCodeFromCustomScripts(customScriptsTable)
    : [];

  const content = formatExtractedCodeForLLM(scripts, workflows, customScripts);
  return content;
}

import { describeRoute } from 'hono-openapi';
import { FlexibeeService, NotFoundError } from '@repo/client-flexibee';
import { flexibeeAuth, Role } from '../config/user.js';
import { validator } from 'hono-openapi/zod';
import {
  abraIdParamSchema,
  companyParamSchema,
  healthcheckDescription,
  inventoryDescription,
  receivedOrderDescription,
  receivedOrderDetailDescription,
  winstromReceivedOrderSchema,
} from '../schemas/flexibee.schema.js';

export default function createFlexibeeV1Handler(flexibeeService: FlexibeeService) {
  const app = new Hono();

  app.get('/', describeRoute(healthcheckDescription), async (c) => {
    try {
      const response = await flexibeeService.isApiWorking('miragecz');
      c.status(response[0] ? 200 : 500);
      return c.json({ flexibee: response[0] });
    } catch (error) {
      if (error instanceof Error) {
        return c.json({ error: error.message }, 500);
      }
      return c.json({ error: 'An unexpected error occurred' }, 500);
    }
  });

  app.post(
    '/received-order/:company',
    describeRoute(receivedOrderDescription),
    validator('json', winstromReceivedOrderSchema),
    validator('param', companyParamSchema),
    flexibeeAuth(Role.CLIENT),
    async (c) => {
      const { company } = c.req.valid('param');
      try {
        const response = await flexibeeService.receivedOrder(company, c.req.valid('json'));
        return c.json(response);
      } catch (error) {
        if (error instanceof Error) {
          return c.json({ error: error.message }, 500);
        }
        return c.json({ error: 'An unexpected error occurred' }, 500);
      }
    }
  );

  app.get(
    '/received-order/:company/:abraId',
    describeRoute(receivedOrderDetailDescription),
    validator('param', abraIdParamSchema),
    flexibeeAuth(Role.CLIENT),
    async (c) => {
      const { company, abraId } = c.req.valid('param');
      try {
        const response = await flexibeeService.receivedOrderDetail(company, abraId);
        return c.json(response);
      } catch (error) {
        if (error instanceof NotFoundError) {
          return c.json({ error: error.message }, 404);
        }
        if (error instanceof Error) {
          return c.json({ error: error.message }, 500);
        }
        return c.json({ error: 'An unexpected error occurred' }, 500);
      }
    }
  );

  app.get(
    '/inventory/:company',
    describeRoute(inventoryDescription),
    validator('param', companyParamSchema),
    flexibeeAuth(Role.CLIENT),
    async (c) => {
      const { company } = c.req.valid('param');

        return c.json(response);

          return c.json({ error: error.message }, 500);
        }

import { basicAuth, getEnvOrFail, nodeEnv } from '@repo/core-api';

export enum Role {
  ADMIN,
  BACKEND,
  CLIENT,
}

export interface User {
  username: string;
  password: string;
  companies: string[];
  roles: Role[];
}

const users = getUsers(nodeEnv);

export const flexibeeAuth = (permissionNeeded: Role) => {
  return basicAuth({
    verifyUser: (username, password, c) => {
      const company = c.req.param().company;
      if (company == null) {
        return false;
      }

      const user = users.find(
        (u) => u.username === username && u.password === password && u.companies.includes(company)
      );

      if (!user) {
        return false;
      }

      return hasAccess(permissionNeeded, user);
    },
  });
};

function hasAccess(permissionNeeded: Role, user: User): boolean {
  return user.roles.findIndex((it) => it.valueOf() <= permissionNeeded.valueOf()) >= 0;
}

function getUsers(env: string): User[] {
  switch (env) {
    case 'production': {
      return [
        {
          username: 'synapto',
          password: getEnvOrFail('USER_SYNAPTO_PASSWORD'),
          companies: ['miragecz', 'miragesk'],
          roles: [Role.ADMIN],
        },
        {
          username: 'miragecz',
          password: getEnvOrFail('USER_MIRAGECZ_PASSWORD'),
          companies: ['miragecz'],
          roles: [Role.BACKEND],
        },
        {
          username: 'mirageCzClient',
          password: getEnvOrFail('USER_MIRAGECZ_CLIENT_PASSWORD'),
          companies: ['miragecz'],
          roles: [Role.CLIENT],
        },
        {
          username: 'miragesk',
          password: getEnvOrFail('USER_MIRAGESK_PASSWORD'),
          companies: ['miragesk'],
          roles: [Role.BACKEND],
        },
        {
          username: 'mirageskClient',
          password: getEnvOrFail('USER_MIRAGESK_CLIENT_PASSWORD'),
          companies: ['miragesk'],
          roles: [Role.CLIENT],

      ];
    }
    case 'development': {
      return [
        {
          username: 'synapto',
          password: getEnvOrFail('USER_SYNAPTO_PASSWORD'),
          companies: ['miragecz', 'mirageczTest'],
          roles: [Role.ADMIN],
        },
      ];
    }
    case 'test': {
      return [
        {
          username: 'test',

    default: {
      throw new Error(`Unknown environment ${env}. Can't load users.`);
    }
  }

  async createAlert(alert: CreateJiraAlertRequest): Promise<JiraAlertResponse> {
    const mode: string = getEnvOrFail('JIRA_ALERTS_MODE');
    switch (mode) {
      case 'log':
        defaultLogger.info('Alert logged (Jira alerts disabled via JIRA_ALERTS_MODE=log)', alert);
        return {
          result: 'ok',
          requestId: 'disabled',
          took: 1,
        };
      case 'production':
        return this.doCreateAlert(alert);
      default:
        defaultLogger.error('Unknown mode for Jira alerts', { mode });

    }

    if (!alert.alias) {
      defaultLogger.warn('No alias provided. This might lead to duplicate alerts.', alert);
    }

    try {
      const response = await fetch('https://api.atlassian.com/jsm/ops/integration/v2/alerts', {
        method: 'POST',
        headers: this.headers(),
        body: JSON.stringify(alert),
      });

      if (!response.ok) {
        const errorText = await response.text();
        throw new Error(
          `Failed to create alert: ${response.status} ${response.statusText} - ${errorText}`
        );
      }

      const res = (await response.json()) as JiraAlertResponse;
      defaultLogger.info('Alert created successfully', {
        id: res.requestId,
        took: res.took,
        alias: alert.alias,
      });

      return res;
    } catch (error) {
      defaultLogger.error('Error creating Jira alert', error);
      throw error;
    }
  }

  private headers() {
    return {
      'Content-Type': 'application/json',
      Accept: 'application/json',
      Authorization: `GenieKey ${getEnvOrFail('JIRA_ALERTS_API_KEY')}`,

        {
          method: 'GET',
          headers: this.headers(),
        }
      );
      const res = (await response.json()) as ClickupTaskList;
      if (res.tasks.length > 0) {
        return res.tasks[0];
      }
      return null;
    }
  }

  private async doCreateAlert(alert: CreateClickupAlertRequest): Promise<ClickupTask> {
    try {
      const response = await fetch(`https://api.clickup.com/api/v2/list/${this.listId}/task`, {

        id: res.id,
        alias: alert.name,
      });

    } catch (error) {
      defaultLogger.error('Error creating Clickup alert', error);

    }
  }

      Accept: 'application/json',

import type { LogEntry } from './types.js';
import slug from 'slug';

export function createDeduplicationKey(logEntry: LogEntry): string {
  // Extract a simplified error source identifier
  const source = logEntry.source || '';

  // Get a simplified name of the affected component
  let component = '';
  if (logEntry.subject) {
    // For API endpoints, extract just the table name if possible
    if (
      source === 'API' &&
      (logEntry.subject.includes('/tables/') || logEntry.subject.includes('tables/'))
    ) {
      // Handle both formats: "/tables/tableName" and "tables/tableName"
      const match = logEntry.subject.match(/(?:\/)?tables\/([^/]+)/);
      component = match ? match[1] : '';
    } else {
      // For workflows and other sources, use the full subject or table
      component = logEntry.subject;
    }
  } else if (logEntry.table) {
    component = logEntry.table;
  }

  // Try to extract an error type from the detail
  let errorIdentifier = '';
  if (logEntry.detail) {
    // Try to extract a meaningful error type from the detail message
    const exceptionMatch = logEntry.detail.match(/Exception of type '([^']+)' was thrown/);
    if (exceptionMatch && exceptionMatch[1]) {
      // Just get the exception name without the namespace
      const parts = exceptionMatch[1].split('.');
      errorIdentifier = parts[parts.length - 1];
    } else {
      // Look for other common error patterns
      if (logEntry.detail.includes('access denied') || logEntry.detail.includes('not authorized')) {
        errorIdentifier = 'AccessDenied';
      } else if (logEntry.detail.includes('Bad value type')) {
        errorIdentifier = 'BadValueType';
      } else if (logEntry.detail.includes('UnhandledPromiseRejection')) {
        errorIdentifier = 'UnhandledPromiseRejection';
      } else {
        errorIdentifier = 'GenericError';
      }
    }
  }

  // Combine components and filter out empty values
  const parts = [logEntry.appId, slug(source), slug(component), errorIdentifier].filter(
    (part) => part !== ''
  );

  // If key is short and a hash is present, add the hash to the end
  if (parts.length === 1 && logEntry.hash) {
    parts.push(logEntry.hash);
  }

  // Join with double colon as separator
  return parts.join('::');
}

import slug from 'slug';
import { defaultLogger } from '@repo/core-api';
import type { TabidooApp } from '@repo/client-tabidoo/types/apps';
import type { TabidooTableDataFilterFieldOperator } from '@repo/client-tabidoo/types/tables';
import type { LogEntry, TabidooLogFields } from './types.js';
import type { CreateJiraAlertRequest } from '../jiraAlerts/types.js';
import { createDeduplicationKey } from './tabidooKey.js';
import type { TabidooConfig } from '../config.js';
import type JiraAlertsClient from '../jiraAlerts/JiraAlertsClient.js';
import type ClickupAlertsClient from '../clickupAlerts/ClickupAlertsClient.js';
import type { CreateClickupAlertRequest } from '../clickupAlerts/types.js';

export class TabidooLogProcessor {
  private readonly instances: TabidooConfig[];
  private readonly useJira: boolean;
  private readonly jiraAlertsClient: JiraAlertsClient;
  private readonly useClickup: boolean;
  private readonly clickupAlertsClient: ClickupAlertsClient;

  constructor(
    instances: TabidooConfig[],
    useJira: boolean,
    jiraAlertsClient: JiraAlertsClient,
    useClickup: boolean,
    clickupAlertsClient: ClickupAlertsClient
  ) {
    this.instances = instances;
    this.useJira = useJira;
    this.jiraAlertsClient = jiraAlertsClient;
    this.useClickup = useClickup;
    this.clickupAlertsClient = clickupAlertsClient;
  }

  async job() {
    try {
      defaultLogger.debug('Starting log downloader job', {
        instances: this.instances.map((instance) => instance.id),
      });
      for (const instance of this.instances) {
        defaultLogger.debug('Downloading logs from Tabidoo apps', {
          id: instance.id,
        });

        const apps = await instance.client.apps.all();

        for (const app of apps) {
          try {
            await this.processLogsFromApp(instance.id, app, instance);
          } catch (error) {
            if (error instanceof Error) {
              defaultLogger.error('Error downloading logs', {
                id: instance.id,
                error: error.message,
                appId: app.id,
                appName: app.internalName,
              });
            } else {
              defaultLogger.error('Error downloading logs', {
                id: instance.id,
                error: 'Unknown error',
                appId: app.id,
                appName: app.internalName,
              });
            }
          }
        }
      }
    } catch (error) {
      if (error instanceof Error) {
        defaultLogger.error('Error running log downloader job', {
          error: error.message,
        });
      } else {
        defaultLogger.error('Error running log downloader job', {
          error: 'Unknown error',
        });
      }
    }
  }

  private async processLogsFromApp(instanceId: string, app: TabidooApp, instance: TabidooConfig) {
    // Get logs
    const logFilter = {
      filter: [
        {
          field: 'type',
          operator: 'eq' as TabidooTableDataFilterFieldOperator,
          value: 'Error',
        },
        {
          field: 'x_created',
          operator: 'gte' as TabidooTableDataFilterFieldOperator,
          value: new Date(Date.now() - 24 * 60 * 60 * 1000).toISOString(),
        },
        {
          field: 'internalNotificationSent',
          operator: 'eq' as TabidooTableDataFilterFieldOperator,
          value: 'false',
        },
      ],
    };
    const logItems = await instance.client.tables.dataWithFilter<TabidooLogFields>(
      app.id,
      'systemLog',
      logFilter
    );
    if (logItems.length === 0) {
      return;
    }

    defaultLogger.debug('Log items', {
      id: instance.id,
      logItems: logItems,
      appId: app.id,
      appName: app.internalName,
    });

    // Send logs to Jira Alerts
    for (const logItem of logItems) {
      await this.processLogItem(instanceId, app, logItem.fields);
    }

    // Mark as processed
    const updatePayload = logItems.map((item) => {
      return {
        id: item.id,
        fields: {
          internalNotificationSent: true,
        },
      };
    });
    await instance.client.tables.bulk.updatePartial(app.id, 'systemLog', updatePayload);
  }

  private async processLogItem(instanceId: string, app: TabidooApp, logItem: TabidooLogFields) {
    const logEntry = {
      appId: app.id,
      appName: app.name,
      instanceId: instanceId,
      create: logItem.x_created,
      type: logItem.type,
      subject: logItem.subject,
      source: logItem.source,
      table: logItem.table,
      user: logItem.user,
      detail: logItem.detail,
      hash: logItem.hash,
    };

    if (this.useJira) {
      const alert = this.createJiraAlertRequest(logEntry);
      await this.jiraAlertsClient.createAlert(alert);
    }
    if (this.useClickup) {
      const alert = this.createClickupErrorEmail(logEntry);
      await this.clickupAlertsClient.createAlert(alert, createDeduplicationKey(logEntry));
    }
  }

  createJiraAlertRequest(logEntry: LogEntry): CreateJiraAlertRequest {
    const component = logEntry.table || logEntry.source || '';
    const includeSubject = !component.includes('Webhook');
    const message = `Tabidoo | ${logEntry.appName} | ${component}${includeSubject && logEntry.subject ? ` | ${logEntry.subject}` : ''}`;
    const description = this.createDescription(logEntry);
    const tags = ['tabidoo', 'production', slug(logEntry.appName)];

    // Create the full alert payload
    return {
      message,
      alias: createDeduplicationKey(logEntry),
      description,
      tags,
      priority: 'P2',
      source: 'alert-bridge',
      details: {
        appName: slug(logEntry.appName),
      },
    };
  }

  createClickupErrorEmail(logEntry: LogEntry): CreateClickupAlertRequest {
    const component = logEntry.table || logEntry.source || '';
    const includeSubject = !component.includes('Webhook');

    return {
      name: `Tabidoo | ${logEntry.appName} | ${component}${includeSubject && logEntry.subject ? ` | ${logEntry.subject}` : ''}`,
      description: this.createDescription(logEntry),
      tags: ['tabidoo', 'production', slug(logEntry.appName), createDeduplicationKey(logEntry)],
      priority: 2,
    };
  }

  createDescription(logEntry: LogEntry): string {
    return `
      Error in ${logEntry.appName}
      ${logEntry.instanceId ? `https://${logEntry.instanceId}/app/${logEntry.appId}` : ''}

      ${logEntry.source ? `Source: ${logEntry.source}` : ''}

      ${logEntry.subject ? `Subject: ${logEntry.subject}` : ''}
      ${logEntry.type ? `Type: ${logEntry.type}` : ''}
      ${logEntry.create ? `Time: ${logEntry.create}` : ''}

      Error Details:
      ${logEntry.detail || 'No details available'}
    `;
  }

  async getApps(): Promise<Record<string, { id: string; name: string }[]>> {
    const apps: Record<string, { id: string; name: string }[]> = {};
    for (const instance of this.instances) {
      try {
        const instanceApps = await instance.client.apps.all();
        apps[instance.id] = instanceApps.map((app) => {
          return {
            id: app.id,
            name: app.name,
          };
        });
      } catch (error) {
        defaultLogger.error('Errror getting apps', {
          id: instance.id,
          error: error instanceof Error ? error.message : 'Unknown error',
        });
      }
    }
    return apps;
  }

// Base interfaces for Make.com emails
export interface MakeEntity {
  title: string;
  url: string;
}

// Common properties shared across email types
interface MakeEmailBase {
  timestamp: Date;
  type: MakeEmailType;
}

// Specific email types for discriminated union
export enum MakeEmailType {
  WARNING = 'warning',
  ERROR = 'error',
  STOPPED = 'stopped',
  INVITATION = 'invitation',
  UNKNOWN = 'unknown',
}

// Email with organization/scenario/team
interface MakeScenarioEmailBase extends MakeEmailBase {
  organization: MakeEntity;
  scenario: MakeEntity;
  scenarioId?: number;
  team: MakeEntity;
}

// Specific email types with discriminated union pattern
export interface MakeWarningEmail extends MakeScenarioEmailBase {
  type: MakeEmailType.WARNING;
  details: string[];
}

export interface MakeErrorEmail extends MakeScenarioEmailBase {
  type: MakeEmailType.ERROR;
  details: string[];
}

export interface MakeStoppedEmail extends MakeScenarioEmailBase {
  type: MakeEmailType.STOPPED;
  reason: string;
}

export interface MakeInvitationEmail extends MakeEmailBase {
  type: MakeEmailType.INVITATION;
  acceptInvitationUrl: string;
  body: string;
}

export interface MakeUnknownEmail extends MakeEmailBase {
  type: MakeEmailType.UNKNOWN;
  body: string;
  subject: string;
}

// Union type for all email types with proper discrimination
export type MakeEmail =
  | MakeWarningEmail
  | MakeErrorEmail
  | MakeStoppedEmail
  | MakeInvitationEmail
  | MakeUnknownEmail;

// Interface for the incoming email JSON
interface IncomingEmail {
  subject: string;
  text: string;
  timestamp: Date;
}

/**
 * Parse an entity from text that might span multiple lines
 * Format is either:
 * " * Entity Name: Entity Title [URL]" or
 * " * Entity Name: Entity Title
 *   [URL]"
 */
function parseEntity(text: string, entityName: string): MakeEntity | null {
  // First try to find the entity line
  const regex = new RegExp(
    `\\*\\s+${entityName}:\\s+(.*?)(?:\\s+\\[(https?:\\/\\/[^\\]]+)\\]|$)`,
    'm'
  );
  const match = text.match(regex);

  if (!match) return null;

  const title = match[1].trim();

  // If URL is on the same line
  if (match[2]) {
    return {
      title,
      url: match[2],
    };
  }

  // If URL is on the next line, we need to look for it
  const titleLineIndex = text.indexOf(match[0]);
  if (titleLineIndex === -1) return { title, url: '' };

  // Get the text right after the title line to look for URL
  const nextLineText = text.substring(titleLineIndex + match[0].length);
  const nextLines = nextLineText.split('\n');

  // Look for a URL on the next line
  if (nextLines.length > 0) {
    const urlMatch = nextLines[0].trim().match(/^\s*\[(https?:\/\/[^\]]+)\]/);
    if (urlMatch && urlMatch[1]) {
      return {
        title,
        url: urlMatch[1],
      };
    }
  }

  // Return just the title if no URL was found
  return {
    title,
    url: '',
  };
}

/**
 * Extract details from email body
 * Extracts detail items that typically come after "The details of the warnings are:" or "Details:"
 */
function extractDetails(text: string): string[] {
  const details: string[] = [];

  // Try to find the details section
  const detailsStartIndex = Math.max(
    text.indexOf('The details of the warnings are:'),
    text.indexOf('Details:')
  );

  if (detailsStartIndex === -1) return details;

  // Get the text after the details heading
  const detailsSection = text.substring(detailsStartIndex);

  // Split the text into lines
  const lines = detailsSection.split('\n');

  // Look for lines that start with " * " which indicate detail items
  for (let i = 1; i < lines.length; i++) {
    const line = lines[i].trim();
    if (line.startsWith('*')) {
      // Extract the detail text without the " * " prefix
      const detail = line.substring(1).trim();
      if (
        detail &&
        !detail.startsWith('Scenario:') &&
        !detail.startsWith('Organization:') &&
        !detail.startsWith('Team:') &&
        !detail.startsWith('Reason:')
      ) {
        details.push(detail);
      }
    } else if (line.startsWith('Need any help?')) {
      // Stop processing when we reach the footer
      break;
    }
  }

  return details;
}

/**
 * Extract the reason from a stopped scenario email
 */
function extractReason(text: string): string {
  const reasonMatch = text.match(/\*\s+Reason:\s+(.*?)$/m);
  return reasonMatch ? reasonMatch[1].trim() : '';
}

/**
 * Extract invitation URL from invitation email
 */
function extractInvitationUrl(text: string): string {
  const urlMatch = text.match(/Accept invitation\s+\[(https?:\/\/[^\]]+)\]/);
  return urlMatch ? urlMatch[1].trim() : '';
}

/**
 * Extract scenario ID from scenario URL
 */
function extractScenarioId(url: string): number | undefined {
  const match = url.match(/\/scenarios\/(\d+)\//);
  const id = match ? match[1] : '';
  return id ? parseInt(id) : undefined;
}

/**
 * Parse the common scenario-based properties (organization, scenario, team)
 */
function parseScenarioEmailBase(
  email: IncomingEmail,
  timestamp: Date
): Omit<MakeScenarioEmailBase, 'type'> {
  const text = email.text;

  // Extract entities using the full text to handle multi-line entities
  const organization = parseEntity(text, 'Organization') || { title: '', url: '' };
  const scenario = parseEntity(text, 'Scenario') || { title: '', url: '' };
  const team = parseEntity(text, 'Team') || { title: '', url: '' };

  // Extract scenario ID from the scenario URL
  const scenarioId = extractScenarioId(scenario.url);

  return {
    timestamp,
    organization,
    scenario,
    team,
    scenarioId,
  };
}

/**
 * Parse the email JSON and return the appropriate typed email object
 */
export function parseEmail(email: IncomingEmail): MakeEmail {
  try {
    // Parse the JSON string to object
    const timestamp = new Date(email.timestamp);

    // Identify email type by subject and parse accordingly
    if (
      email.subject.includes('warnings in scenario') ||
      email.subject.includes('warning in scenario')
    ) {
      return parseWarningEmail(email, timestamp);
    } else if (email.subject.includes('errors in') || email.subject.includes('error in')) {
      return parseErrorEmail(email, timestamp);
    } else if (email.subject.includes('has been stopped')) {
      return parseStoppedEmail(email, timestamp);
    } else if (email.subject.includes('invited to join')) {
      return parseInvitationEmail(email, timestamp);
    } else {
      return {
        type: MakeEmailType.UNKNOWN,
        timestamp,
        body: email.text,
        subject: email.subject,
      };
    }
  } catch (error) {
    defaultLogger.error('Error parsing email', error);
    return {
      type: MakeEmailType.UNKNOWN,
      timestamp: new Date(),
      subject: email.subject,
      body: email.text,
    };
  }
}

/**
 * Parse warning email
 */
function parseWarningEmail(email: IncomingEmail, timestamp: Date): MakeWarningEmail {
  const baseProperties = parseScenarioEmailBase(email, timestamp);
  const details = extractDetails(email.text);

  return {
    ...baseProperties,
    type: MakeEmailType.WARNING,
    details,
  };
}

/**
 * Parse error email
 */
function parseErrorEmail(email: IncomingEmail, timestamp: Date): MakeErrorEmail {
  const baseProperties = parseScenarioEmailBase(email, timestamp);
  const details = extractDetails(email.text);

  return {
    ...baseProperties,
    type: MakeEmailType.ERROR,
    details,
  };
}

/**
 * Parse stopped scenario email
 */
function parseStoppedEmail(email: IncomingEmail, timestamp: Date): MakeStoppedEmail {
  const baseProperties = parseScenarioEmailBase(email, timestamp);
  const reason = extractReason(email.text);

  return {
    ...baseProperties,
    type: MakeEmailType.STOPPED,
    reason,
  };
}

/**
 * Parse invitation email
 */
function parseInvitationEmail(email: IncomingEmail, timestamp: Date): MakeInvitationEmail {
  const acceptInvitationUrl = extractInvitationUrl(email.text);

  return {
    timestamp,
    type: MakeEmailType.INVITATION,
    acceptInvitationUrl,
    body: email.text,
  };
}

export const isWarningEmail = (email: MakeEmail): email is MakeWarningEmail =>
  email.type === MakeEmailType.WARNING;

export const isErrorEmail = (email: MakeEmail): email is MakeErrorEmail =>
  email.type === MakeEmailType.ERROR;

export const isStoppedEmail = (email: MakeEmail): email is MakeStoppedEmail =>
  email.type === MakeEmailType.STOPPED;

export const isInvitationEmail = (email: MakeEmail): email is MakeInvitationEmail =>
  email.type === MakeEmailType.INVITATION;

export const isUnknownEmail = (email: MakeEmail): email is MakeUnknownEmail =>
  email.type === MakeEmailType.UNKNOWN;

    this.jiraAlertsClient = jiraAlertsClient;
    this.useClickup = useClickup;
    this.clickupAlertsClient = clickupAlertsClient;

    this.client = new Make(
      getEnvOrFail('MAKE_API_KEY'),

    );

  async processMakeEmail(email: ParsedMail): Promise<void> {
    defaultLogger.debug('Processing body', { subject: email.subject, text: email?.text });

    const parsedEmail = parseEmail({
      subject: email.subject ?? '',
      text: email?.text ?? '',
      timestamp: email.date ?? new Date(),
    });
    if (this.useJira) {
      if (isErrorEmail(parsedEmail)) {
        const jiraErrorEmail = await this.createJiraErrorEmail(parsedEmail);
        await this.sendJiraAlert(jiraErrorEmail);
      } else if (isWarningEmail(parsedEmail)) {
        const jiraWarningEmail = await this.createJiraWarningEmail(parsedEmail);
        await this.sendJiraAlert(jiraWarningEmail);
      } else if (isStoppedEmail(parsedEmail)) {
        const jiraStoppedEmail = await this.createJiraStoppedEmail(parsedEmail);
        await this.sendJiraAlert(jiraStoppedEmail);
      } else if (isInvitationEmail(parsedEmail)) {
        defaultLogger.info('Invitaion email received', { email: parsedEmail });
      } else {
        defaultLogger.warn('Unknown email received', { email: parsedEmail });
      }
    }
    if (this.useClickup) {
      if (isErrorEmail(parsedEmail)) {
        const clickupErrorEmail = await this.createClickupErrorEmail(parsedEmail);
        const deduplicationKey = createDeduplicationKey(parsedEmail);
        await this.sendClickupAlert(clickupErrorEmail, deduplicationKey);
      } else if (isWarningEmail(parsedEmail)) {
        const clickupWarningEmail = await this.createClickupWarningEmail(parsedEmail);
        const deduplicationKey = createDeduplicationKey(parsedEmail);
        await this.sendClickupAlert(clickupWarningEmail, deduplicationKey);
      } else if (isStoppedEmail(parsedEmail)) {
        const clickupStoppedEmail = await this.createClickupStoppedEmail(parsedEmail);
        const deduplicationKey = createDeduplicationKey(parsedEmail);
        await this.sendClickupAlert(clickupStoppedEmail, deduplicationKey);
      } else if (isInvitationEmail(parsedEmail)) {
        defaultLogger.info('Invitaion email received', { email: parsedEmail });
      } else {
        defaultLogger.warn('Unknown email received', { email: parsedEmail });
      }
    }
  }

    }

import {
  type MakeWarningEmail,
  type MakeErrorEmail,
  isWarningEmail,
  type MakeStoppedEmail,
  isErrorEmail,
} from './emailParser.js';

function extractIdFromUrl(url: string): string {
  const scenarioMatch = url.match(/\/scenarios\/(\d+)/);
  if (scenarioMatch && scenarioMatch[1]) {
    return scenarioMatch[1];
  }

  const teamMatch = url.match(/\/(\d+)\/team/);
  if (teamMatch && teamMatch[1]) {
    return teamMatch[1];
  }

  const orgMatch = url.match(/\/organization\/(\d+)/);
  if (orgMatch && orgMatch[1]) {
    return orgMatch[1];
  }

  return '';
}

export function createDeduplicationKey(
  email: MakeWarningEmail | MakeErrorEmail | MakeStoppedEmail
): string {
  const scenarioId = extractIdFromUrl(email.scenario.url);
  const teamId = extractIdFromUrl(email.team.url);
  const orgId = extractIdFromUrl(email.organization.url);

  let emailType: string;
  if (isWarningEmail(email)) {
    emailType = 'warning';
  } else if (isErrorEmail(email)) {
    emailType = 'error';
  } else {
    emailType = 'stopped';
  }

  return `make-${emailType}-${scenarioId}-${teamId}-${orgId}`;
}

import {
  SMTPServer,
  type SMTPServerAddress,
  type SMTPServerDataStream,
  type SMTPServerSession,
} from 'smtp-server';
import { simpleParser, type ParsedMail } from 'mailparser';
import { defaultLogger } from '@repo/core-api';
import type { MakeEmailProcessor } from './MakeEmailProcessor.js';

const allowedAddresses = ['@make.com', '@integromat.com', '@eu1.make.celonis.com'];

export class MakeSMTPServer {
  private server?: SMTPServer;
  private readonly makeEmailProcessor: MakeEmailProcessor;

  constructor(makeEmailProcessor: MakeEmailProcessor) {
    this.makeEmailProcessor = makeEmailProcessor;
  }

  startSMTPServer(port: number = 25) {
    const makeEmailProcessor = this.makeEmailProcessor;

    this.server = new SMTPServer({
      disabledCommands: ['AUTH', 'STARTTLS', 'XCLIENT', 'XFORWARD'],

      onMailFrom(
        address: SMTPServerAddress,
        session: SMTPServerSession,
        callback: (err?: Error | null) => void
      ): void {
        if (allowedAddresses.some((allowed) => address.address.endsWith(allowed))) {
          callback();
        } else {
          defaultLogger.warn(`Recepient address not allowed`, { address: address.address });
          callback(new Error('Recepient address not allowed'));
        }
      },

      onData(
        stream: SMTPServerDataStream,
        session: SMTPServerSession,
        callback: (err?: Error | null) => void
      ): void {
        simpleParser(stream)
          .then((email: ParsedMail) => {
            defaultLogger.info('Email received', {
              from: email.from?.text,
              subject: email.subject,
            });
            makeEmailProcessor.processMakeEmail(email).then(() => callback());
          })
          .catch((err: Error) => {
            defaultLogger.error('Error parsing email', err);
            callback(new Error('Error parsing email'));
          });
      },

      // Set maximum message size (10 MB)
      size: 10 * 1024 * 1024,
    });

    this.server.listen(port, () => {
      defaultLogger.info('SMTP Server running', { port });
    });
    this.server.on('error', (err: Error) => {
      defaultLogger.error('SMTP Server error', err);
    });
  }

  stopSMTPServer() {
    if (!this.server) {
      return;
    }
    this.server.close(() => {
      defaultLogger.info('Stopping SMTP Server');
    });
  }
}

const shutdownHandlers: Array<() => Promise<void>> = [];

export function addOnShutdown(handler: () => Promise<void>) {

export function createServer(options?: Partial<ServerOptions>) {

    serverOptions = { ...serverOptions, ...options };
  }

  app.use(requestLogger(defaultLogger.info));

  });

  process.on('SIGINT', async () => {
    defaultLogger.info('SIGINT signal received.');
    for (const handler of shutdownHandlers) {
      await handler();
    }
    process.exit(0);
  });

  return app;
}

export function startServer(app: Hono<BlankEnv, BlankSchema, '/'>) {
  defaultLogger.info('Starting server', serverOptions);

  if (serverOptions?.openApiOptions) {
    app.get('/openapi/openapi.json', openAPISpecs(app, serverOptions.openApiOptions));

    app.get(
      '/openapi/ui',
      apiReference({
        baseServerURL: `http://${serverOptions.hostname}:${serverOptions.port}`,
        spec: { url: '/openapi/openapi.json' },
      })
    );
  }

  return serve({
    fetch: app.fetch,
    port: serverOptions.port,
    hostname: serverOptions.hostname,
  });
}

// Original source:
// https://github.com/honojs/hono/blob/main/src/middleware/logger/index.ts
import type { Context, HonoRequest, MiddlewareHandler } from 'hono';
import { getConnInfo } from '@hono/node-server/conninfo';
import { getEnv } from '@repo/core-env';

enum LogPrefix {
  Outgoing = '-->',
  Incoming = '<--',
}

const humanize = (times: string[]) => {
  const [delimiter, separator] = [',', '.'];

  const orderTimes = times.map((v) => v.replace(/(\d)(?=(\d\d\d)+(?!\d))/g, '$1' + delimiter));

  return orderTimes.join(separator);
};

const time = (start: number) => {
  const delta = Date.now() - start;
  return humanize([delta < 1000 ? delta + 'ms' : Math.round(delta / 1000) + 's']);
};

// eslint-disable-next-line @typescript-eslint/no-explicit-any
type LoggerFunc = (message: string, ...meta: any[]) => void;
type LoggerFilterFunc = (req: HonoRequest) => boolean;

// eslint-disable-next-line @typescript-eslint/no-unused-vars
const allowAll = (req: HonoRequest): boolean => true;
const notUptimeMonitoring = (req: HonoRequest): boolean => {
  const userAgent = req.header('User-Agent') || '';
  return !userAgent.includes('Uptime-Kuma');
};

const notDockerHealthcheck = (c: Context): boolean => {
  const ip = getRemoteIpAddress(c);
  const { url } = c.req;
  const path = url.slice(url.indexOf('/', 8));
  return !(ip === '127.0.0.1' && path === '/');
};

export const requestLogger = (
  fn: LoggerFunc,
  filterFn: LoggerFilterFunc = allowAll
): MiddlewareHandler => {
  return async function logger(c, next) {
    if (filterFn(c.req) && notUptimeMonitoring(c.req) && notDockerHealthcheck(c)) {
      const { method, url } = c.req;

      const path = url.slice(url.indexOf('/', 8));
      const ip = getRemoteIpAddress(c);

      fn(`${LogPrefix.Incoming} ${method} ${path}`, {
        ip,
      });

      const start = Date.now();

      await next();

      fn(`${LogPrefix.Outgoing} ${method} ${path} ${c.res.status} ${time(start)}`);
    } else {
      await next();
    }
  };
};

function getRemoteIpAddress(c: Context) {
  // server info is not available when running tests
  const env: string = getEnv('NODE_ENV', 'development');
  if (env === 'test') return '127.0.0.1';
  return getConnInfo(c)?.remote.address;
}

import type { WinstromReceivedOrderRequest } from './types/order.js';
import {
  type AccountingPeriod,
  type AccountingPeriodResponse,
  transformAccountingPeriod,
} from './types/accounting-period.js';
import type { FlexibeeResponse } from './types/flexibee-response.js';
import type { Inventory } from './types/inventory.js';

export interface FlexibeeConfig {
  [key: string]: FlexibeeCompanyConfig;
}

export interface FlexibeeCompanyConfig {
  username: string;
  password: string;
  baseurl: string;
  company: string;
  mainWarehouseId: number;
  secondaryWarehouseId: number;
}

export class FlexibeeClient {
  readonly config: FlexibeeConfig;
  accountingPeriods: Map<string, AccountingPeriod[]> = new Map();

  constructor(config: FlexibeeConfig) {
    this.config = config;

    Object.keys(config).forEach((company) => {
      this.accountingPeriods.set(company, []);
      this.syncAccountingPeriods(company).then(() => {});
    });
  }

  async isApiWorking(company: string): Promise<[boolean, Error?]> {
    const config = this.getCompanyConfig(company);
    const path = '/skladova-karta?limit=1&detail=id';
    try {
      await this.get(config, path);
      return [true];
    } catch (e) {
      if (e instanceof Error) {
        return [false, e];
      }
      throw e;
    }
  }

  async receivedOrder(company: string, data: WinstromReceivedOrderRequest): Promise<object> {
    const config = this.getCompanyConfig(company);
    const path = '/objednavka-prijata/?dry-run=true';
    return this.post(config, path, data);
  }

  async receivedOrderDetail(
    company: string,
    abraId: number
  ): Promise<WinstromReceivedOrderRequest> {
    const config = this.getCompanyConfig(company);
    const path = `/objednavka-prijata/${abraId}`;
    return this.get(config, path);
  }

  async getCurrentAccountingPeriod(company: string): Promise<AccountingPeriod> {
    const findActivePeriod = (periods: AccountingPeriod[]) => {
      return periods.find((p) => isPeriodActive(p));
    };
    const isPeriodActive = (period: AccountingPeriod) => {
      const now = new Date();
      return now >= period.platiOdData && now <= period.platiDoData;
    };

    let companyAccountingPeriods = this.accountingPeriods.get(company);
    if (companyAccountingPeriods === undefined) {
      throw new Error('Unknown company');
    }
    let currentPeriod = findActivePeriod(companyAccountingPeriods);

    if (!currentPeriod) {
      companyAccountingPeriods = await this.syncAccountingPeriods(company);
      currentPeriod = findActivePeriod(companyAccountingPeriods);
      if (!currentPeriod) {
        throw new Error('No active accounting period found for current date');
      }
    }
    return currentPeriod;
  }

  private async syncAccountingPeriods(company: string): Promise<AccountingPeriod[]> {
    const config = this.getCompanyConfig(company);
    const path = '/ucetni-obdobi?detail=full';
    const response = (await this.get(config, path)) as FlexibeeResponse<AccountingPeriodResponse>;

    // Transform the response to convert strings to actual types
    const accountingPeriods = response.winstrom['ucetni-obdobi'].map(transformAccountingPeriod);
    this.accountingPeriods.set(company, accountingPeriods);
    return accountingPeriods;
  }

  async getInventory(company: string): Promise<Inventory> {
    const config = this.getCompanyConfig(company);
    const accountingPeriod = await this.getCurrentAccountingPeriod(company);

    const path = `/skladova-karta/(sklad in(${config.mainWarehouseId},${config.secondaryWarehouseId}) and ucetObdobi eq ${accountingPeriod.id})?detail=full&limit=0&add-row-count=true`;
    const response = (await this.get(config, path)) as FlexibeeResponse<Inventory>;
    return response.winstrom;
  }

  private async get<T>(config: FlexibeeCompanyConfig, path: string): Promise<FlexibeeResponse<T>> {
    const uri = config.baseurl + config.company + path;
    const response = await fetch(uri, {
      method: 'GET',
      headers: this.getHeaders(config),
    });

    const statusCode = response.status;
    if (response.ok) {
      return (await response.json()) as FlexibeeResponse<T>;
    } else if (statusCode === 404) {
      throw new NotFoundError('Record not found in Flexibee');
    } else if (statusCode === 401) {
      throw new Error('Failed to authorize Flexibee, invalid credentials');
    } else {
      throw new Error('Failed to call Flexibee');
    }
  }

  private async post<T>(
    config: FlexibeeCompanyConfig,
    path: string,
    data: WinstromReceivedOrderRequest
  ): Promise<FlexibeeResponse<T>> {
    const uri = config.baseurl + config.company + path;
    const response = await fetch(uri, {
      method: 'POST',
      headers: this.getHeaders(config),
      body: JSON.stringify(data),
    });

    const statusCode = response.status;
    if (response.ok) {
      return (await response.json()) as FlexibeeResponse<T>;
    } else if (statusCode === 401) {
      throw new Error('Failed to authorize Flexibee, invalid credentials');
    } else {
      throw new Error('Failed to call Flexibee');
    }
  }

  private getCompanyConfig(company: string): FlexibeeCompanyConfig {
    return this.config[company];
  }

  private getHeaders(config: FlexibeeCompanyConfig) {
    return {
      Authorization:
        'Basic ' + Buffer.from(config.username + ':' + config.password).toString('base64'),
      'Content-Type': 'application/json',
      Accept: 'application/json',
    };
  }
}

/**
 * Root structure for accounting periods response
 */
export interface AccountingPeriodResponse {
  '@version': string;
  'ucetni-obdobi': RawAccountingPeriod[];
}

/**
 * Raw API response for accounting period before transformation
 */
export interface RawAccountingPeriod {
  /** Unique identifier of the accounting period */
  id: string;
  /** Last update timestamp */
  lastUpdate: string;
  /** Start date of the accounting period */
  platiOdData: string;
  /** End date of the accounting period */
  platiDoData: string;
  /** Description of the accounting period */
  popis: string;
  /** Additional notes */
  poznam: string;
  /** Code of the accounting period (usually year) */
  kod: string;
  /** Indicates if closing entries have been changed */
  zmenaZaver: string;
  /** Indicates if there are revaluation errors */
  chybaPreceneni: string;
  /** Year for the sequence */
  rokProRadu: string;
}

/**
 * Interface representing a single accounting period in Flexibee after transformation
 */
export interface AccountingPeriod
  extends Omit<
    RawAccountingPeriod,
    'zmenaZaver' | 'chybaPreceneni' | 'platiOdData' | 'platiDoData'
  > {
  /** Indicates if closing entries have been changed */
  zmenaZaver: boolean;
  /** Indicates if there are revaluation errors */
  chybaPreceneni: boolean;
  /** Start date of the accounting period */
  platiOdData: Date;
  /** End date of the accounting period */
  platiDoData: Date;
}

/**
 * Parses date string in format "YYYY-MM-DD+HH:mm" to Date object
 * @param dateString Date string from API
 * @param isEndDate If true, sets time to end of day (23:59:59.999)
 * @returns Date object
 */
function parseDateWithTimezone(dateString: string, isEndDate: boolean = false): Date {
  const [datePart] = dateString.split('+');
  const date = new Date(datePart);

  if (isEndDate) {
    date.setHours(23, 59, 59, 999);
  } else {
    date.setHours(0, 0, 0, 0);
  }

  return date;
}

/**
 * Transforms raw API response to typed accounting period
 * @param raw Raw accounting period data from API
 * @returns Transformed accounting period with proper types
 */
export function transformAccountingPeriod(raw: RawAccountingPeriod): AccountingPeriod {
  return {
    ...raw,
    zmenaZaver: raw.zmenaZaver === 'true',
    chybaPreceneni: raw.chybaPreceneni === 'true',
    platiOdData: parseDateWithTimezone(raw.platiOdData, false),
    platiDoData: parseDateWithTimezone(raw.platiDoData, true),
  };
}

export default function normalizeApiUrl(url: string): string {
  const cleanUrl = url.replace(/\/+$/, "");

  const apiPath = "/api";
  if (cleanUrl.endsWith(apiPath)) {
    return cleanUrl;
  }

  const apiPattern = /\/api$/;
  const baseUrl = cleanUrl.replace(apiPattern, "");

  return `${baseUrl}${apiPath}`;
}

import type { TabidooData, TabidooDataArray } from '../types/base.js';

export type ApiLoggerFn = (message: string, data?: object) => void;

export default class APIClient {
  private customHeaders: Map<string, string> = new Map();

  constructor(
    private readonly baseUrl: string,
    private readonly token: string,
    private readonly logger: ApiLoggerFn = () => {}
  ) {}

  withHeader(key: string, value: string): this {
    this.customHeaders.set(key, value);
    return this;
  }

  async get<R>(url: string): Promise<R> {
    return this.fetchJson(url, 'GET');
  }

  async getOne<R>(url: string): Promise<R> {
    const res = (await this.fetchJson(url, 'GET')) as TabidooData<R>;
    return res.data;
  }

  async getArray<R>(url: string): Promise<R[]> {
    const res = (await this.fetchJson(url, 'GET')) as TabidooDataArray<R>;
    return res.data;
  }

  async post<B, R>(url: string, body: B): Promise<R> {
    return this.fetchJson(url, 'POST', body);
  }

  async postOne<B, R>(url: string, body: B): Promise<R> {
    const res = (await this.fetchJson(url, 'POST', body)) as TabidooData<R>;
    return res.data;
  }

  async postArray<B, R>(url: string, body: B): Promise<R[]> {
    const res = (await this.fetchJson(url, 'POST', body)) as TabidooDataArray<R>;
    return res.data;
  }

  async putOne<B, R>(url: string, body: B): Promise<R> {
    const res = (await this.fetchJson(url, 'PUT', body)) as TabidooData<R>;
    return res.data;
  }

  async putArray<B, R>(url: string, body: B): Promise<R[]> {
    const res = (await this.fetchJson(url, 'PUT', body)) as TabidooDataArray<R>;
    return res.data;
  }

  async patch<B, R>(url: string, body: B): Promise<R> {
    return this.fetchJson(url, 'PATCH', body);
  }

  async patchOne<B, R>(url: string, body: B): Promise<R> {
    const res = (await this.fetchJson(url, 'PATCH', body)) as TabidooData<R>;
    return res.data;
  }

  async patchArray<B, R>(url: string, body: B): Promise<R[]> {
    const res = (await this.fetchJson(url, 'PATCH', body)) as TabidooDataArray<R>;
    return res.data;
  }

  private async fetchJson<T>(url: string, method: string, body?: unknown): Promise<T> {
    const headers = new Headers();
    headers.append('Content-Type', 'application/json');
    headers.append('Authorization', `Bearer ${this.token}`);
    this.customHeaders.forEach((value, key) => {
      headers.append(key, value);
    });

    const options: RequestInit = {
      method,
      headers,
    };

    if (body) {
      options.body = JSON.stringify(body);
    }

    this.logger(`Tabidoo API call: ${method} ${this.baseUrl}${url}`, {
      method,
      url,
      body,
      headers: headers,
    });

    const response = await fetch(`${this.baseUrl}${url}`, options);
    this.customHeaders.clear();

    if (!response.ok) {
      const errorText = await response.text();

      if (errorText) {
        try {
          const errorJson = JSON.parse(errorText);
          this.logger(`APIClient error, http=${response.status}`, errorJson);
        } catch (e: unknown) {
          if (e instanceof Error) {
            throw new Error(
              `APIClient unknown error, http=${response.status}, exception=${e?.message}`
            );
          } else {
            throw new Error(`APIClient unknown error, http=${response.status}`);
          }
        }
        throw new Error(`APIClient request failed with status ${response.status} ${errorText}`);
      } else {
        this.logger(`APIClient error, http=${response.status}`, { message: 'Empty response body' });
        throw new Error(`APIClient request failed with status ${response.status}`);
      }
    }

    return response.json() as T;
  }
}

export interface ExtractedCode {
  appId: string;
  appName: string;
  fragments: ExtractedCodeFragment[];
}

export interface ExtractedCodeFragment {
  table: string;
  fieldName: string;
  codeJs: string;
  codeTs: string;
}

export function extractExecutableCode(appStructure: TabidooAppFull): ExtractedCode {
  const extractedCode: ExtractedCode = {
    appId: appStructure.id,
    appName: `${appStructure.name} (${appStructure.internalName})`,
    fragments: [],
  };

  appStructure.tables.forEach((table) => {
    const tableName = table.internalNameApi;

    // Check fields for executable code
    table.items.forEach((field) => {
      if (field.type === 'calculatedfield' && field.metadata?.script?.jsScript) {
        extractedCode.fragments.push({
          table: tableName,
          fieldName: field.name,
          codeJs: field.metadata.script.jsScript,
          codeTs: field.metadata.script.tsScript || '',
        });
      }
      if (field.type === 'buttonform' && field.metadata?.script?.jsScript) {
        extractedCode.fragments.push({
          table: tableName,
          fieldName: field.name,
          codeJs: field.metadata.script.jsScript,
          codeTs: field.metadata.script.tsScript || '',
        });
      }
      if (field.type === 'freehtmlinput' && field.metadata?.freeHtmlInitScript?.jsScript) {
        extractedCode.fragments.push({
          table: tableName,
          fieldName: field.name,
          codeJs: field.metadata.freeHtmlInitScript.jsScript,
          codeTs: field.metadata.freeHtmlInitScript.tsScript || '',
        });
      }
    });

    // Check table-level scripts
    if (table.scripts) {
      table.scripts.forEach((script) => {
        if (script.jsScript) {
          extractedCode.fragments.push({
            table: tableName,
            fieldName: script.name, // Use script name for table-level scripts
            codeJs: script.jsScript,
            codeTs: script.tsScript || '',
          });
        }
      });
    }
  });

  return extractedCode;
}

export interface ExtractedWorkflowCodeFragment {
  workflow: string;
  triggers: string;
  codeJs: string;
  codeTs: string;
}

export function extractExecutableCodeFromWorkflows(
  workflows: TabidooTableData<TabidooWorkflowTableFields>[]
): ExtractedWorkflowCodeFragment[] {
  const extractedCode: ExtractedWorkflowCodeFragment[] = [];
  workflows.forEach((workflow) => {
    workflow.fields.definition.steps.forEach((step) => {
      if (step.type === 'jsScript' && step.data && 'script' in step.data) {
        extractedCode.push({
          workflow: workflow.fields.name,
          triggers: JSON.stringify(workflow.fields.definition.triggers),
          codeJs: step.data?.script.runableSript || '',
          codeTs: step.data?.script.writtenTypeScript || '',
        });
      }
    });
  });
  return extractedCode;
}

interface ExtractedCustomScriptCodeFragment {
  name: string;
  namespace: string;
  interface: string;
  dts: string;
  script: string;
}

export function extractExecutableCodeFromCustomScripts(
  customScripts: TabidooTableData<TabidooCustomScriptFields>[]
): ExtractedCustomScriptCodeFragment[] {
  const extractedCode: ExtractedCustomScriptCodeFragment[] = [];

  customScripts.forEach((table: TabidooTableData<TabidooCustomScriptFields>) => {
    const fields: TabidooCustomScriptFields = table.fields;
    extractedCode.push({
      name: fields.name,
      namespace: fields.namespace,
      interface: fields.interface,
      dts: fields.dts.runableSript,
      script: fields.script.runableSript,
    });
  });

  return extractedCode;
}

export function formatExtractedCodeForLLM(
  extractedCode: ExtractedCode,
  workflows?: ExtractedWorkflowCodeFragment[],
  customScripts?: ExtractedCustomScriptCodeFragment[]
): string {
  const sections = [
    `# Application Code Analysis\n\n**Application ID:** ${extractedCode.appId}\n**Application Name:** ${extractedCode.appName}\n\n`,
  ];

  extractedCode.fragments.forEach((codeBlock, index) => {
    sections.push(`## Code Block ${index + 1}\n`);
    sections.push(`**Table:** ${codeBlock.table}`);
    sections.push(`**Field:** ${codeBlock.fieldName}\n`);
    sections.push(`\`\`\`typescript\n`);
    sections.push(codeBlock.codeTs);
    sections.push('```\n');
  });

  if (workflows && workflows.length > 0) {
    sections.push(`## Workflow Code\n\n`);
    workflows.forEach((workflow, index) => {
      sections.push(`### Workflow ${index + 1}: ${workflow.workflow}\n`);
      sections.push(`**Triggers:** ${workflow.triggers}\n`);
      sections.push(`\`\`\`typescript\n`);
      sections.push(workflow.codeTs);
      sections.push('```\n');
    });
  }

  if (customScripts && customScripts.length > 0) {
    sections.push(`## Custom Scripts\n\n`);
    customScripts.forEach((script, index) => {
      sections.push(`### Custom Script ${index + 1}: ${script.name}\n`);
      sections.push(`**Namespace:** ${script.namespace}\n`);
      sections.push(`**Interface:** ${script.interface}\n`);
      sections.push(`\`\`\`typescript\n`);
      sections.push(script.script);
      sections.push('```\n');
    });
  }

  return sections.join('\n');
}

import * as winston from 'winston';
import type { Logger } from 'winston';
import { getEnv, isEnvDefined, nodeEnv } from '@repo/core-env';
import { buildGCPTransport, buildJsonConsoleTransport, buildLocalTransport } from './transports.js';

export type { Logger };

const isLocal = ['test', 'development'].includes(nodeEnv);

export function createLogger(): Logger {
  const logLevel = getEnv<string>('LOG_LEVEL', 'debug');

  if (isLocal && !isEnvDefined('LOG_DEST')) {
    return createLocalLogger(logLevel);
  } else {
    return createProductionLogger(logLevel);
  }
}

function createLocalLogger(logLevel: string) {
  const logger = winston.createLogger({
    level: logLevel,
    transports: [buildLocalTransport()],
  });
  logger.info('Local logger initialized', {
    logLevel,
  });
  return logger;
}

function createProductionLogger(logLevel: string) {
  const logDest: string[] = getEnv<string>('LOG_DEST', 'json-console').split(',');
  const transports: winston.transport[] = [];

  logDest.forEach((dest) => {
    switch (dest) {
      case 'json-gcp':
        transports.push(buildGCPTransport());
        break;
      case 'json-console':
        transports.push(buildJsonConsoleTransport());
        break;
      default:
        throw new Error(`Unknown log destination: ${dest}`);
    }
  });

  const logger = winston.createLogger({
    level: logLevel,
    transports,
  });

  logger.info('Logger initialized', {
    logDestination: logDest,
    logLevel,
  });

  return logger;
}

import * as winston from 'winston';

import { getEnv, getEnvOrFail } from '@repo/core-env';

const timestampFormat = winston.format.timestamp({
  format: 'YYYY-MM-DD HH:mm:ss',
});

export function buildLocalTransport(): winston.transport {
  return new winston.transports.Console({
    format: winston.format.combine(
      timestampFormat,
      winston.format.simple(),
      winston.format.colorize({ all: true })
    ),
  });
}

export function buildJsonConsoleTransport(): winston.transport {
  return new winston.transports.Console({
    format: winston.format.combine(
      timestampFormat,
      winston.format.uncolorize(),
      winston.format.json()
    ),
  });
}

export function buildGCPTransport(): winston.transport {
  return new LoggingWinston({
    format: winston.format.combine(
      timestampFormat,
      winston.format.uncolorize(),
      winston.format.json()
    ),
    logName: getEnvOrFail<string>('GCP_LOG_NAME'),
    projectId: getEnvOrFail<string>('GCP_PROJECT_ID'),
    resource: {
      type: getEnv('GCP_RESOURCE_TYPE', 'gce_instance'),
      labels: {
        instance_id: getEnvOrFail('GCP_INSTANCE_ID'),
        zone: getEnvOrFail('GCP_ZONE'),
      },
    },
  });
}

import EnvironmentLoader from './env/EnvironmentLoader.js';

type EnvValue = string | number | boolean;

export function loadEnv(): string[] {
  const loader = new EnvironmentLoader();
  return loader.load();
}

export const isEnvDefined = (key: string): boolean => key in process.env;

export const getEnv = <T extends EnvValue>(
  key: string,
  defaultValue?: T,
  transform?: (value: string) => T
): T => {
  const value = process.env[key];

  if (value === undefined) {
    if (defaultValue === undefined) {
      throw new Error(
        `Environment variable ${key} is not defined and no default value was provided`
      );
    }
    return defaultValue;
  }

  if (transform) {
    return transform(value);
  }

  if (typeof defaultValue === 'number') {
    return Number(value) as T;
  }

  if (typeof defaultValue === 'boolean') {
    return (value.toLowerCase() === 'true') as T;
  }

  return value as T;
};

export const getEnvOrFail = <T extends EnvValue>(
  key: string,
  transform?: (value: string) => T
): T => {
  const value = process.env[key];

  if (value === undefined) {
    throw new Error(`Environment variable ${key} is not defined`);
  }

  if (transform) {
    return transform(value);
  }

  return value as T;
};

export const nodeEnv = getEnv('NODE_ENV', 'development');

# File name: synapto-services-main/packages/core-env/src/env/EnvironmentLoader.ts
import dotenv from 'dotenv';
import path from 'path';
import fs from 'fs';

type NodeEnv = 'development' | 'production' | 'test';

export default class EnvironmentLoader {
  private directory: string;
  private nodeEnv: NodeEnv;

  constructor() {
    this.directory = process.cwd();
    this.nodeEnv = (process.env.NODE_ENV as NodeEnv) || 'development';
  }

  private getEnvFilePath(filename: string): string {
    return path.resolve(this.directory, filename);
  }

  private fileExists(filepath: string): boolean {
    return fs.existsSync(filepath);
  }

  private loadEnvFile(filepath: string): boolean {
    if (this.fileExists(filepath)) {
      const result = dotenv.config({ path: filepath, override: true });
      if (result?.error) {
        throw new Error(`Failed to load environment file ${filepath}: ${result.error}`);
      }
      return true;
    }
    return false;
  }

  public load(): string[] {
    // Load files in order of precedence (lower files override higher files)
    const envFiles = ['.env', `.env.${this.nodeEnv}`, '.env.local'];
    const loadedFiles: string[] = [];

    for (const envFile of envFiles) {
      const result = this.loadEnvFile(this.getEnvFilePath(envFile));
      if (result) {
        loadedFiles.push(envFile);
      }
    }
    return loadedFiles;
  }
}
